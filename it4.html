Instant Recovery

With virtualization solutions users can boot a backup of a system volume into a virtual machine within milliseconds

even if a VM (Virtual Machine) is TB-sized for temporary failover or permanent recovery.

Simple Recovery Tests

Makes it easy to test backups by letting you boot a backup image into a virtual machine to make sure a restored

system will function properly. That provides you with a fast, easy, and reliable approach for regularly verifying that

your backups are on track.

Virtual Sandboxing

Providing you with a virtual sandbox to test new applications. Just launch a virtual machine of the production

system’s latest backup file and install the new application on the virtual machine. That makes it possible for you to

evaluate the software’s performance within your actual IT environment without putting your production system at

risk.
